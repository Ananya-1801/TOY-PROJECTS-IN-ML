# End-to-End Machine Learning Project: Student Placement Prediction

## 📌 Project Overview
This project implements an **end-to-end Machine Learning classification pipeline** to predict whether a university student gets **placed or not** based on two key academic indicators:
- **IQ**
- **CGPA**

The project demonstrates the complete ML workflow starting from raw data to model training, evaluation, visualization, and persistence.

---

## 🎯 Problem Statement
Universities and recruiters often analyze academic performance indicators to estimate placement outcomes.  
The objective of this project is to **predict student placement status (Placed / Not Placed as values 1/0 )** using:
- IQ score
- CGPA achieved

This is modeled as a **binary classification problem**.

---

## 🧠 Machine Learning Approach
- **Model Used:** Logistic Regression  
- **Type:** Supervised Learning (Binary Classification)

---

## 🛠️ Technologies & Libraries Used
- **Python**
- **NumPy** – numerical operations  
- **Pandas** – data handling  
- **Matplotlib** – data visualization  
- **Scikit-learn** – preprocessing, model training & evaluation  
- **MLxtend** – decision boundary visualization  
- **Pickle** – model serialization
